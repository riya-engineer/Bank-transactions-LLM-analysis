## 🚀 Project Overview  

💡 This project uses **LangChain + Groq LLM** to analyze bank transaction datasets using **natural language queries**.  
📊 Instead of writing SQL or Pandas code, you can simply *ask questions* and get insights instantly.  

---

## ⚡ Features  

✅ Ask questions like:  
- "How many transactions are there?"  
- "Describe the dataset"  
- "Show trends in transactions"  

✅ AI-powered data exploration  
✅ Automated insights generation  
✅ Works directly with CSV datasets  
✅ Built using modern LLM stack

## ⚙️ How It Works  

1️⃣ Load dataset using **Pandas**  
2️⃣ Connect to **Groq LLM (LLaMA 3)**  
3️⃣ Create CSV agent using **LangChain**  
4️⃣ Ask questions in natural language  
5️⃣ Get AI-generated insights  

---

## 🧠 Example Queries  

```python
agent.invoke("How many rows and columns are in this dataset?")
agent.invoke("Describe columns")
agent.invoke("Show transaction trends")
````
